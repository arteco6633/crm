# Инструкция по настройке CRM системы

## Шаг 1: Установка зависимостей

```bash
npm install
```

## Шаг 2: Настройка переменных окружения

Создайте файл `.env` в корне проекта (скопируйте из `env.example`):

```bash
cp env.example .env
```

Файл `.env` уже содержит правильные настройки для вашего Supabase проекта.

## Шаг 3: Настройка базы данных в Supabase

1. Откройте Supabase Dashboard: https://uaenlkqvnaavithpelvs.supabase.co
2. Перейдите в раздел **SQL Editor**
3. Скопируйте содержимое файла `supabase/schema.sql`
4. Вставьте в SQL Editor и выполните запрос

Это создаст три таблицы:
- `clients` - для хранения клиентов
- `deals` - для хранения сделок  
- `tasks` - для хранения задач

## Шаг 4: Запуск приложения

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Шаг 5: Первый запуск Git

Если вы хотите загрузить код в репозиторий:

```bash
git add .
git commit -m "Initial commit: CRM система с Supabase"
git branch -M main
git push -u origin main
```

## Проверка работы

После запуска сервера откройте браузер и перейдите на http://localhost:3000

Вы должны увидеть интерфейс CRM системы с тремя вкладками:
- Клиенты
- Сделки
- Задачи

Попробуйте добавить первого клиента, чтобы проверить работу системы!
